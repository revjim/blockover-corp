Replace `blockover-logo.svg` with your actual Blockover Corp logo.

The logo is currently a placeholder. Upload your logo file here and ensure it's named:
- `blockover-logo.svg` (preferred for scalability)
- Or update the reference in `/components/Navigation.tsx` if using a different format/name

## Recommended Logo Specifications

- Format: SVG (scalable) or PNG (high resolution)
- Size: 40x40px minimum for display, but provide higher resolution
- Background: Transparent
- Colors: Should work on white background
